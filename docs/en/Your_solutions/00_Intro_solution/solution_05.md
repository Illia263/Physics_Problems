### Problem 5: Trigonometry

**Given:**

- Magnitude of the vector: $|\vec{A}| = 15$
- Angle with the horizontal axis: $\theta = 60^\circ$

**Goal:** Calculate the horizontal ($A_x$) and vertical ($A_y$) components.

**Step 1: Calculate the horizontal component ($A_x$)**
The horizontal component is adjacent to the angle, so we use the cosine function:

$$A_x = |\vec{A}| \cos(\theta)$$

$$A_x = 15 \cdot \cos(60^\circ)$$

Since $\cos(60^\circ) = \frac{1}{2}$ (or $0.5$):

$$A_x = 15 \cdot 0.5$$

$$A_x = 7.5$$

**Step 2: Calculate the vertical component ($A_y$)**
The vertical component is opposite to the angle, so we use the sine function:

$$A_y = |\vec{A}| \sin(\theta)$$

$$A_y = 15 \cdot \sin(60^\circ)$$

Since $\sin(60^\circ) = \frac{\sqrt{3}}{2}$:

$$A_y = 15 \cdot \frac{\sqrt{3}}{2}$$

$$A_y = 7.5\sqrt{3}$$

_(which is approximately 12.99)_

**Final Answer:**

- **Horizontal component:** $A_x = 7.5$
- **Vertical component:** $A_y = 7.5\sqrt{3}$
