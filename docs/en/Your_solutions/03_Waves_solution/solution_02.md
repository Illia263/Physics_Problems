### Problem 2: String Harmonics

**Given:**
* Length of string ($L$): $64 \text{ cm} = 0.64 \text{ m}$
* Fundamental frequency ($f_1$): $330 \text{ Hz}$
* Harmonic mode: Fundamental (1 antinode)

**Goal:** Find the speed of the wave on this string ($v$).

**Step 1: Relate string length to wavelength**
For a string fixed at both ends, the fundamental frequency (the first harmonic) occurs when the length of the string corresponds to exactly half of a wavelength.
$$L = \frac{\lambda}{2}$$
Rearranging to find the wavelength ($\lambda$):
$$\lambda = 2L$$

**Step 2: Calculate the wavelength**
Substitute the length of the string in meters:
$$\lambda = 2 \cdot 0.64$$
$$\lambda = 1.28 \text{ m}$$

**Step 3: Calculate the wave speed**
Using the fundamental wave equation relating speed, frequency, and wavelength:
$$v = f \cdot \lambda$$
Substitute the known frequency and the calculated wavelength:
$$v = 330 \cdot 1.28$$
$$v = 422.4 \text{ m/s}$$

---

**Final Answer:**
The speed of the wave on the guitar string is **$422.4 \text{ m/s}$**.
