### Problem 7: Standing Wave Modes

**Given:**
* Length of string ($L$): $80 \text{ cm} = 0.8 \text{ m}$
* Number of antinodes ($n$): $4$

**Goal:** Find the wavelength of this wave ($\lambda$).

**Step 1: Relate string length to wavelength**
For a standing wave on a string fixed at both ends, the string length is divided into an integer number of half-wavelengths. The number of antinodes (loops) corresponds directly to this integer $n$ (the harmonic number).
$$L = n \left(\frac{\lambda}{2}\right)$$

**Step 2: Rearrange the formula to solve for wavelength**
Multiply both sides by 2 and divide by $n$:
$$\lambda = \frac{2L}{n}$$

**Step 3: Calculate the wavelength**
Substitute the known length and number of antinodes:
$$\lambda = \frac{2 \cdot 0.8}{4}$$
$$\lambda = \frac{1.6}{4}$$
$$\lambda = 0.4 \text{ m}$$

---

**Final Answer:**
The wavelength of the standing wave is **$0.4 \text{ m}$** (or $40 \text{ cm}$).
